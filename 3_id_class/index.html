id e class

CLASS

existem duas caracteristicas principais do class

  • é possivel utilizar o mesmo class em varios elementos diferentes
  • é possivbel utilizar varias classes no mesmo elemento

ID

existem duas caracteristicas principais do ID que que sao contrastantes com as caracteristicas do class:

  • nao é possivel utilizar o mesmo ID em varios elementos diferentes.
  • nao é possivel utilizar varios IDS no mesmo elemnto

o ID funciona como CPF, nao podem exister dois iguals e a mesma pessoa nao ter mais de dois CPFs
